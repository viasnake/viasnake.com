# yomiage bot とは

チャンネルに書き込まれたテキストを読み上げる Discord Bot です。

## 注意事項

- 本 Bot を利用した場合の損害について一切の責任を負いません。
- 本 Bot は事前に予告なく内容等の改変や削除、サービス停止を行う可能性があります。

## よくある質問

- 音声設定を変更できない。
  - 一度ボットをボイスチャンネルに接続してみて、何かを読み上げさせてから、設定してみてください。
  - 一度も読み上げたことがないと、ユーザがまだ認識されていない可能性があります。
- 音声設定を他サーバでも使いたい。
  - 音声設定はすべてのサーバで共有されます。
- なんかおかしい。
  - 連絡ください。
