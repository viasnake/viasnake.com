# 利用方法

ボットを `/connect` でボイスチャンネルに接続後、テキストチャンネルに書き込んだテキストが読み上げられます。

## 音声設定を変更する場合

現在、以下の設定を変更可能です。

- 音声のピッチ（デフォルト: `0.0`）
- 音声の速度（デフォルト: `1.0`）
- 音声の言語（デフォルト: `ja-JP`）
- 音声の種類（デフォルト: `ja-JP-Standard-A`）

音声のピッチを変更する場合、`/setpitch` を実行してください。
ピッチは -20 から 20 の間で選択可能です。

音声の速度を変更する場合、`/setspeakingrate` を実行してください。
速度は 0.25 から 4.0 の間で選択可能です。

音声の言語を変更する場合、`/setlanguage` を実行してください。
表示されたメニューから言語を選択可能です。

音声の種類を変更する場合、`/setvoice` を実行してください。
表示されたメニューから音声を選択可能です。

## 読み上げるチャンネルを指定する場合

読み上げるチャンネルで `/setchannel` を実行してください。
そのチャンネルのみが読み上げ対象のチャンネルとなります。

すべてのチャンネルで読み上げたい場合は `/setchannel global` を実行してください。
サーバ上のすべてのチャンネルが読み上げ対象のチャンネルとなります。
