# コマンド

- `/connect`
  - 今いるボイスチャンネルに接続する。
- `/disconnect`
  - ボイスチャンネルから切断する。
- `/ping`
  - レイテンシーを表示する。
- `/setpitch`
  - 音声のピッチを設定する。
- `/setspeakingrate`
  - 音声の速度を設定する。
- `/setlanguage`
  - 音声の言語を設定する。
- `/setvoice`
  - 音声の種類を設定する。
- `/setchannel`
  - 読み上げるチャンネルを設定する。
